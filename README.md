# Healthcare Analysis Dashboard (Power BI + SQL)

An end-to-end Healthcare Analysis Dashboard built for hospital administration teams to track doctors, patients, revenue, expenses, and operational insights in a single interactive view.
This project integrates SQL for ETL and Power BI for powerful visual analytics.

# Project Overview

The objective of this project is to enable data-driven decision-making in hospitals and clinics by providing a consolidated view of operational and financial performance.
Primary users:
Hospital Administrators
Finance Teams
Operations & Management Teams

The dashboard eliminates the need to browse large datasets by offering clear, visual insights for daily and strategic decision-making.

# Key Features

1. Doctors Analytics

Track patients handled per doctor to evaluate workload balance.
View average charges per patient and total revenue per doctor.
Identify top-performing doctors by contribution and efficiency.

2. Financial Performance

Analyze monthly revenue vs expenses with trend visuals.
Evaluate doctor-wise monthly cost and operational expenditure.
Support budgeting, forecasting, and cost-optimization strategies.

3. Patient Management

Explore complete patient details in a clean, filterable table.
Analyze types and number of tests conducted per patient.
Review medication details and cost breakdown for billing accuracy.

# Tech Stack

SQL – Data extraction, cleaning, and transformation
Power BI – Interactive dashboard creation and visualization
Data Modeling – Relationships and calculated measures for analysis

# How to Use

Clone or download this repository.
Open the Healthcare_Analysis_Dashboard.pbix file in Power BI Desktop.
(Optional) Update data source connections to your own database or sample data.
Refresh the model and explore the dashboards using slicers and filters.
